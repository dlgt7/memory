# 内存映像网计算器 - 手机专用版

专为PLC工程师打造的「手机端」内存映像网地址快速计算工具  
再也不用开电脑、开GX Works、翻手册了！

**手机浏览器直接打开 → 添加至主屏幕 → 秒变独立App**

### 在线使用地址（请星标收藏）：
https://dlgt7.github.io/memory

### 功能一览
1. 模拟量 R ↔ 内存映像网地址（D区）快速互算（×2规律）
2. 数字量 M ↔ 内存映像网地址 + 位号 快速互算（8点一字规律）
3. 支持三菱、欧姆龙、西门子等主流PLC通用规则
4. 纯离线运行，零广告，纯前端，无需联网

### 四大计算模式
- 模拟量：已知起始D和R → 算终止D地址
- 模拟量：已知起始D和终止D → 算终止R地址
- 数字量：已知起始D和起始M → 算终止D地址+位号
- 数字量：已知起始D、终止D、起始M、位号 → 算终止M地址

### 特色
- 手机竖屏完美适配
- 支持回车键快速计算
- 输入框颜色提示（红=内存地址，蓝=R，绿=M）
- 可添加到手机桌面，像原生App一样使用
- 完全开源免费，放心使用
